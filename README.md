# Kejar Mimpi API

Provided API to Kejar Mimpi App

## Dependency
* Go Language (Golang)
* Setting GOPATH
* Beego Framework Golang
* Postgresql

### Installing

run this command:

```
glide install or glide update
```
then

```
bee run -downdoc=true -gendoc=true
```
open your browser 
```
http://127.0.0.1:8080/swagger/
```
testing local ==> video
```
https://www.youtube.com/watch?v=TCcldd7AnzY
```
